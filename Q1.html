Purpose of var() Function :-

1.Reusability :

Css variables allow us to store values that we can reuse across ouy stylesheet. The vara() function lets us reference these values wherever needed, reducing redudancy and making our code cleaner.

2.Consistency :

By Using variables, we can ensure consistent values for colors, spacing, fonts, and more throughout our entire design. Changing a variables's value will automatically update all instance where the variables used.

3.Global Changes :

If we need to make design changes, we can update the variable's value once, and the change will be appliesd to all instances using that variables.
